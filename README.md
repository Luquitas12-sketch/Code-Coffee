## ✨ ¿Qué es Code & Coffee Workspace?

> **Code & Coffee Workspace** es una *Single Page Application (SPA)* desarrollada como proyecto académico para la materia **Paradigmas de la Programación** en la **Universidad Nacional del Este**. Ofrece una experiencia de usuario fluida para la **reserva de espacios de trabajo individual** y **pedidos anticipados de cafetería**, todo desde una interfaz elegante inspirada en la estética del café artesanal.

<div align="center">

```
┌─────────────────────────────────────────────────────────┐
│  🗺️  Mapa de Mesas    →    ☕ Menú     →    🛒 Carrito  │
│       Seleccioná          Agregá tus         Confirmá   │
│       tu lugar            favoritos          el pedido  │
│                                                         │
│              💳 Checkout   →   📦 Live Tracker          │
│           Pagá con tarjeta      Seguí tu pedido         │
│              o en efectivo       en tiempo real         │
└─────────────────────────────────────────────────────────┘
```

## ⚙️ Instalación y Ejecución

> Seguí estos pasos para correr el proyecto en tu máquina local 🖥️

**1. Clonar el repositorio**
```bash
git clone https://github.com/tu-usuario/code-coffee-workspace.git
cd code-coffee-workspace
```

**2. Instalar dependencias**
```bash
npm install
```

**3. Ejecutar en modo desarrollo**
```bash
npm run dev
```

**4. Abrir en el navegador**
```
http://localhost:5173
```

> 💡 **Tip:** Asegurate de tener **Node.js v18+** instalado antes de empezar.
